# HANDOVER — gazer dev session 2026-04-12 (session 4)

## 1. Project Overview

**gazer** — browser-based animated robot eye system. Three.js robot body, 2D eye compositor, WebSocket-driven behavior engine, autonomous mobility. Single `index.html` (~6200 lines), Python scripts for relay/sensors/sim. No build step.

**Stack:** Vanilla JS + Three.js + Canvas 2D (`index.html`), Python (`server.py`, `simulate.py`, `recorder.py`, `replay.py`, etc.)

Current version: **v0.30**

## 4. Key Decisions & Rationale

- **Delta-drift not accumulated-drift:** Microsaccade drift must apply only the *frame delta* (`d.x - prevX`) to `state.gazeX`, not the full accumulated offset. Accumulated offset would cause gaze to walk far from the fixation target over time.
- **File System Access API for save:** Chosen over a server endpoint to avoid changes to `server.py`. Falls back to anchor-download for browsers that don't support it.
- **Visual params only in presets:** Behavior params (blink timing, saccade rates, etc.) are excluded from FACE_DESIGN_PARAMS so A/B comparisons are purely visual, not behavioral.
- **localStorage migration not reset:** Additive migration (only upgrades specific keys) so user's saved slider values aren't lost when defaults change.
- **Collapsed by default:** Face Designs panel starts collapsed to not clutter the default UI for non-design work.

## 5. Pitfalls & Lessons Learned

- **Drift accumulation bug:** Adding `d.x` (the full accumulated Brownian offset) each frame multiplies it by frame count. Only add the per-frame delta. Test by watching `state.gazeX` at the browser console for a few seconds.
- **`fetch_papers.py` UnicodeEncodeError on Windows:** cp1252 console can't encode `\u2010` (Unicode hyphen) in paper titles. Fix: `sys.stdout.reconfigure(encoding='utf-8', errors='replace')` at top of script.
- **`head` Bash command blocked by hook:** Don't use `head -5` for file reading; use `Read` tool with `limit` parameter.
- **All prior invariants still apply** — see `CLAUDE.md` for driveYaw normalization, selectAttentionPOI total===0 guard, Charger exclusion, etc.
